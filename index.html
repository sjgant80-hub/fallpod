What is FallPod?

A sovereign data pod inspired by Tim Berners-Lee's Solid. One unified path-based store that every estate tool — FallMirror, Bloom, FallBrief, whatever comes next — reads from and writes to. No per-tool silos.

Namespace: filesystem-style paths like /bloom/current, /mirror/sessions/2026-07-06, /signatures/.

Auth: keyed to your FallID DID. Encryption derived from your identity.

Backing: OPFS for large blobs (Chrome/Edge), IndexedDB for everything else. Optional FallStore CID references for content-addressed blobs.

Grants: apps request access to path patterns. You approve. You revoke.

Library usage

import { openPod } from 'https://sjgant80-hub.github.io/fallpod/fallpod.js';

const pod = await openPod({ ownerDid: myFallID.did });

await pod.put('/bloom/current', { streak: 47, last: Date.now() });
const cur = await pod.get('/bloom/current');

await pod.grant('fallmirror', ['/mirror/**']);
pod.subscribe('/bloom/', evt => console.log('bloom changed', evt));
